# StockPilot Vue — Inventory & Sales Management Dashboard

StockPilot Vue is a professional Vue 3 inventory and sales management dashboard built with TypeScript, Vite, Pinia, Vue Router, Tailwind CSS, and business-dashboard UI.

It is intentionally designed as an operational warehouse/control-room system, not like the previous React projects. It includes SKU/barcode workflows, product CRUD, stock movement, sales, purchases, suppliers, customers, invoices, low-stock alerts, reports, roles, and settings.
